## Program Overview
The Data Science Diploma is a comprehensive live program covering the complete data science workflow: from data collection and cleaning, through analysis and visualization, to machine learning and big data. You'll work with real datasets and build a portfolio of projects.

## Curriculum
- **Week 1-2:** Python for Data Science & SQL
- **Week 3-4:** Statistics & Exploratory Data Analysis
- **Week 5-6:** Machine Learning & Predictive Modeling
- **Week 7-8:** Data Visualization (Power BI, Tableau, Python)
- **Week 9-10:** Big Data & Cloud Tools
- **Week 11-12:** Capstone & Career Prep

## Projects
- Project 1: End-to-end EDA on real-world dataset
- Project 2: Predictive ML model deployment
- Project 3: Interactive dashboard (Power BI / Tableau)
- Capstone: Full data science project from scratch
