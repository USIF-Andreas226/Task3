## Program Overview
The Full-Stack Web Development Diploma is a comprehensive live program designed to take complete beginners to job-ready full-stack developers. Over 14 weeks, you'll build 5+ real-world projects, master frontend and backend technologies, and graduate with a professional portfolio.

## Curriculum
- **Weeks 1-2:** HTML, CSS, JavaScript Fundamentals
- **Weeks 3-4:** Advanced JavaScript & TypeScript
- **Weeks 5-6:** React.js & Frontend Development
- **Weeks 7-8:** Node.js, Express & REST APIs
- **Weeks 9-10:** Databases (PostgreSQL & MongoDB)
- **Weeks 11-12:** DevOps, Docker & Deployment
- **Weeks 13-14:** Capstone & Career Prep

## Projects
- Project 1: Responsive portfolio website
- Project 2: Interactive web app with React
- Project 3: REST API with Node.js + database
- Project 4: Full-stack application (frontend + backend + DB + deployment)
- Capstone: Complete full-stack application from scratch
